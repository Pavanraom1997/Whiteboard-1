## Drawing Tools

### Pen Tool ‚úèÔ∏è
- Click and drag to draw freehand
- Change color and thickness anytime
- Perfect for handwriting and sketching

### Highlighter üñäÔ∏è
- Like the pen, but with transparency
- Great for marking important areas
- Automatically uses a thicker stroke

### Eraser üßπ
- Removes drawings and objects
- Works by drawing white over content
- Can erase entire objects when selected

### Text Tool üìù
- Click anywhere to place a text box
- Double-click to edit text
- Drag to reposition
- Drag corners to resize

### Shape Tools (Rectangle, Circle, Line)
- Click and drag to create shapes
- All shapes are outlined (no fill)
- Use the select tool to resize after creation

### Select Tool üñ±Ô∏è
- Click objects to select them
- Drag to move objects
- Drag corners to resize
- Press **Delete** or **Backspace** to remove selected objects
- Select multiple objects by dragging a selection box

## Working with Pages

### Managing Pages (Right Sidebar)

**Add a New Page**:
- Click the **"+ Add"** button
- Each page is independent with its own canvas

**Switch Between Pages**:
- Click any page thumbnail to open it
- Current page is highlighted with a blue border

**Duplicate a Page**:
- Hover over a page thumbnail
- Click the **Copy** icon
- Creates an exact copy of the page

**Delete a Page**:
- Hover over a page thumbnail
- Click the **Trash** icon
- You must have at least one page

## File Operations (Top Bar)

### Creating Projects

**New Project**:
1. Click **"New"**
2. Enter a project name
3. Click **"Create"**

### Saving Your Work

**Save Project**:
1. Click **"Save"**
2. Downloads a `.json` file to your computer
3. Keep this file to restore your work later

üí° **Tip**: Save frequently! The app shows "(Unsaved)" next to the project name when there are changes.

### Opening Projects

**Open Existing Project**:
1. Click **"Open"**
2. Select a `.json` file you saved earlier
3. All pages will be restored

### Importing Images

**Import an Image**:
1. Click **"Import"**
2. Select an image file (JPG, PNG, etc.)
3. Image appears on the canvas
4. Use the select tool to move and resize

### Exporting

**Export as PNG**:
1. Click **"Export"**
2. Saves the current page as a PNG image
3. Great for sharing or printing

## Common Questions

**Q: Can I undo a drawing?**  
A: Currently, you can use the eraser or select and delete. Full undo/redo is coming soon!

**Q: Can multiple people use this at the same time?**  
A: Not yet. Real-time collaboration is planned for a future version.

**Q: Where are my projects stored?**  
A: Projects are saved as JSON files on your computer. You control where they're saved.

**Q: Can I import PDFs?**  
A: Not yet, but PDF import with annotation is planned for the next version!

**Q: What file format are projects saved in?**  
A: Projects are saved as `.json` files containing all your pages and drawings.

**Q: Can I change the canvas background color?**  
A: Not yet through the UI, but this feature is coming soon!

## Troubleshooting

**Problem**: Canvas is not responding  
**Solution**: Refresh the page. Your work is auto-saved to each page.

**Problem**: Image import not working  
**Solution**: Make sure the image file is a standard format (JPG, PNG, GIF)

**Problem**: Can't delete an object  
**Solution**: Make sure you're using the Select tool, click the object, then press Delete

**Problem**: Drawing is too thin/thick  
**Solution**: Adjust the stroke width using the slider in the left toolbar

**Problem**: Wrong color selected  
**Solution**: Click a different color in the color palette (left toolbar)
